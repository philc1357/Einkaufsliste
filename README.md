# Einkaufsliste
Diese Anwendung speichert fügt jedes input einem Array hinzu.
Dieses Array wird in einer .json Datei gespeichert.
Anschließend wird für jedes Item im Array ein list-item erzeugt und der Wert ausgegeben.
#
Zusätzlich verfügt diese Anwendung über ein Theme welches als Wert in einer Session gespeichert wird.
Bei jedem Laden der Seite wird dieses Theme abgerufen und entsprechend des Wertes eine Klasse erstellt.
Diese Klasse ändert das Theme und ist über css deklariert.
